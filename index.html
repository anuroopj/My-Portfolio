My Work

Project One

The Apple Store Problem

For this project, I designed an object-oriented program that simulates a simplified version of an app store. The system manages app listings, user accounts, downloads, and app ratings submitted by registered users. This helped me practice organizing data, structuring classes, and applying core OOP concepts in a real-world style scenario.

Project Two

The Vaccine Tracker Problem

This project focused on modeling a vaccination tracking system.The program manages vaccine distribution through registered sites, keeps track of patient records, and generates proof of received vaccine doses. It helped me practice organizing data, designing modular programs, and applying object-oriented programming principles.
